# News Aggregator API

This project is a RESTful API for a personalized news aggregator, developed as part of a **backend development assignment at Airtribe**.  
The goal of the assignment was to design and implement secure REST APIs by following given specifications and ensuring that **all test cases provided by Airtribe pass successfully**.

The API supports user authentication, preference management, and protected access to news data fetched from an external news service.
